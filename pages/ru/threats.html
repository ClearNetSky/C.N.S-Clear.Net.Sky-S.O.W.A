Emotet

Тип: Троян / Ботнет Уровень риска: Высокий Впервые обнаружен: 2014 · всё ещё активен

Описание

Многофункциональный троян, ориентированный на кражу учетных данных и установку дополнительного вредоносного ПО. Использует машинное обучение для обхода традиционных систем защиты.

Цели атаки

Корпоративные сети, финансовые учреждения, системы управления. Распространяется через фишинговые письма и скомпрометированные веб-сайты.

Индикаторы компрометации (IOC)

Пример SHA-256: 0a1b…e9f4 · Подозрительные C2-домены и IP блокируются на уровне DNS в реальном времени. Обращайте внимание на неожиданный запуск PowerShell из макросов Office.

Методы распространения

Фишинговые кампании, эксплуатация уязвимостей, загрузки с скомпрометированных сайтов

Меры по устранению

  1. Обновите антивирусные сигнатуры до последней версии
  2. Заблокируйте указанные IOC в ваших системах защиты
  3. Проведите проверку всех систем на наличие компрометации
  4. Обновите ПО для устранения известных уязвимостей
